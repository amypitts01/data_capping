The graphics below explore some variables in the dataset. The three we are interested in gaining information from are: Author, Quote, and Genre.

Count of Quotes Per Genre

This histogram is showing the number of quotes in each of the 117 genres. The most common genre is age and is closely followed by alone, art, business and amazing. There are close to 1,000 quotes labeled with each of these genre. The generes with the fewest quotes are Mother's Day, St.Patrick's Day, Father's Day, Memorial Day, and Valentine's Day. These genres are very close to the x-axis, so their counts are close to zero. The counts of these quotes are all less than 20 each.

Count of Quotes Per Author

This histogram shows the number of quotes per Author. Because there are 11,173 unique Authors this histogram only shows the top 1,000 Authors. It is clear from this graph that most of these authors write approximately 60 quotes each or less, but the maximum number of quotes an author wrote is just below 140.

Number of Words vs. Characters Per Quote

This scatterplot shows the relationship between the number of words per quote compared to the number of characters per quote. The plot clearly shows that as the number of words increase, the number of character increase and as the number of characters increase, the number of words increases.

Number of Quotes with <=280 Characters

This histogram shows if the Number of Characters in the Quote are less than or equal to 280. If there are 280 characters or less, then the quote is labeled True, but is there are more than 280 characters, the quote is labeled False. The number of characters is important for when the quotes we create are posted to twitter. A tweet can have a maximum length of 280 characters.
