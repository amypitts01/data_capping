Data Cleaning

For this dataset, the data cleaning process was relatively simple. The Quotes, Authors, and Genres were all formatted correctly and there were no null-objects. We had to remove 14,895 quotes, but we were still left with 61,071 unique quotes to use for our model. We also added a column that checks to see if the quote is less than 280 characters, which is the maximum length a Tweet can be, number of words in a quote and number of characters of a quote.
